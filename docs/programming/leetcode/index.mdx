    ### Sort Array/List
    ```java
    // Sort list
    List<Integer> list = new ArrayList<>();
    Collections.sort(list); // ascending
    Collections.sort(list, Collections.reverseOrder()); // descending

    // export to Array
    Integer[] array = list.toArray(new Integer[0]);
    ```

    ### Sort Custom Objects
    ```java
    // Sort by single field
    Arrays.sort(intervals, (a, b) -> a[0] - b[0]);
    
    // Sort by multiple fields
    Arrays.sort(people, (a, b) -> {
        if (a[0] != b[0]) return a[0] - b[0];
        return b[1] - a[1];
    });
    ```

    ### PriorityQueue (Heap)
    ```java
    // Min heap
    PriorityQueue<Integer> minHeap = new PriorityQueue<>();
    
    // Max heap
    PriorityQueue<Integer> maxHeap = new PriorityQueue<>((a, b) -> b - a);
    
    // Custom object heap
    PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[1] - b[1]);
    ```

    ## String Operations

    ### Normal
    ```java
    String s = "hello";
    s.charAt(index); // get character at index
    s.length(); // get length of string
    s.substring(start, end); // get substring from start to end
    s.substring(start, start + length); // get substring from start to start + length
    s.indexOf(substring); // get index of substring
    s.isEmpty(); // check if string is empty
    s.equals(otherString); // check if string is equal to other string
    s.equalsIgnoreCase(otherString); // check if string is equal to other string ignoring case
    s.contains(substring); // check if string contains substring
    s.startsWith(prefix); // check if string starts with prefix
    s.endsWith(suffix); // check if string ends with suffix
    s.replace(oldChar, newChar); // replace old character with new character
    ```

    ### StringBuilder
    ```java
    StringBuilder sb = new StringBuilder();
    sb.append("text");
    sb.reverse();
    sb.toString();
    sb.setCharAt(index, char);
    sb.deleteCharAt(index);
    ```

## Quick Tips
- Use `Arrays.asList()` for fixed-size list
- StringBuilder for string manipulation
- PriorityQueue for k-th element problems
- HashMap for frequency counting and two-sum type problems
